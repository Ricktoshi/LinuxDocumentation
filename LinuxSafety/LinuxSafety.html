Super user.

I recommend setting a separate password for root from your account password. Also, if several people use the same machine, do take the time to see if any of them require access to sudo.

Disable SSH root login.

Edit the sshd_config file. It is the main configuration file for the sshd service. Open The file at: /etc/ssh/. You need to be logged in as root. Look for the line #PermitRootLogin no. and un comment it. If not present, add it. Restart the sshd service: In Debian based distros $ /etc/init.d/sshd restart In Manjaro Arch/Manjaro $ systemctl restart sshd Now if someone wants to hack you via SSH, they would need to know your user ID, password and root password.
